Osobní automatizace pro Windows

Můj AutoHotkey, který drží práci v pohybu.

UsesThis.ahk je můj osobní skript pro AutoHotkey v2. Spouští aplikace, otevírá webové nástroje a zkracuje každodenní úkony ve Windows na pár kláves.

Prohlédnout skript Jak ho používám
  • AutoHotkey v2
  • Windows
  • Veřejný zdrojový kód
UsesThis.ahk 
#Requires AutoHotkey v2.0

; Ctrl + Alt + B → Bluetooth
^!b::{
  Run "ms-settings:bluetooth",, "max"
}
jedna zkratka, jeden úkol Ctrl + Alt + B

01 — Co umí

Méně hledání. Méně klikání. Více hotové práce.

Skript soustřeďuje moje nejčastější digitální návyky do kláves, které mám vždy po ruce.

Aplikace na zkratku

Chrome, VS Code, Word, Excel nebo další nástroje spustím bez hledání v nabídce Start.

Webové nástroje z numpadu

Gmail, Kalendář, Keep, Disk nebo YouTube se otevírají jako samostatné pracovní aplikace.

Windows bez oklik

Bluetooth, zvuk, Správce úloh a další systémová místa mají vlastní rychlou cestu.

02 — AutoHotkey

Automatizace bez zbytečné složitosti.

AutoHotkey je open-source skriptovací jazyk pro Windows. Umožňuje vytvářet vlastní klávesové zkratky, spouštět programy a skládat drobné automatizace, které přesně odpovídají osobnímu způsobu práce.

UsesThis.ahk je moje praktická pracovní vrstva nad Windows. Je osobní, ale veřejně dostupná jako inspirace pro každého, kdo si chce podobně upravit vlastní počítač.

04 — První setkání

Od nenápadné zmínky k vlastnímu systému zkratek.

O AutoHotkey jsem se poprvé dozvěděl koncem roku 2010. Četl jsem tehdy knihy o produktivitě a krátká zmínka v článku Lea Babauty mě přiměla zjistit víc.

Myšlenka byla jednoduchá: počítač nemusí být neměnné prostředí. I malé úpravy mohou každý den ušetřit řadu zbytečných kroků. UsesThis.ahk od té doby roste spolu s tím, jak se mění moje práce.
